# 語聲同行 2.0 — 競賽專題說明

> 飛書 AI 校園挑戰賽 2026 — 開放創新賽道

---

## 專案名稱

**語聲同行 2.0（Voice Companion 2.0）**
— 排灣族語 Voice-to-Voice AI 語伴

## 一句話描述

用 Voice-to-Voice AI 技術為瀕危語言建立可互動的學習夥伴，讓每個排灣族孩子都能用母語跟 AI vuvu（祖父母）對話。

---

## 背景與動機

- 台灣排灣族語被聯合國列為瀕危語言
- 年輕一代面臨母語斷層，缺乏互動學習場景
- 全球 7000+ 種語言中 40% 正在消失，台灣 16 族原住民語言全部瀕危
- 傳統語言教學是單向的（課本 + 錄音），缺乏即時語音互動

## 解決方案

建立一個 **Voice-to-Voice 全鏈路 AI 系統**：

```
用戶說排灣語 → ASR 語音辨識 → LLM 理解與生成 → 雙語回覆
```

核心角色是 **vuvu Maliq** — 一位排灣族 AI 祖母，能夠：
- 用排灣語 + 中文雙語對話
- 識別 12 類排灣語意圖（問候、詢問、否定、文化故事等）
- 溫柔地糾正發音
- 分享排灣族文化知識
- 展示推理過程（Chain-of-Thought），降低幻覺

---

## 技術架構

| 模組 | 技術 | 說明 |
|------|------|------|
| ASR 語音辨識 | Whisper-tiny + LoRA 微調 | 排灣語微調，81.05% 準確率 |
| 意圖識別 | 12 類意圖 + 綴詞法則 | 平均置信度 0.84 |
| 發音校正 | 音韻規則 + Levenshtein | tj→t, m↔v 等排灣語特有音變 |
| LLM 對話 | 智譜 GLM-4-Flash | vuvu Maliq 角色對話 + 117 句語料 |
| 抗幻覺 | Chain-of-Thought | 推理過程展示 + 信心度標注 |
| TTS 語音合成 | XTTS v2 微調（進行中） | +104 排灣語 tokens |
| Web 介面 | Gradio | 錄音 + 文字 + 場景按鈕 |

## 數據

| 項目 | 數量 |
|------|------|
| 真實錄音 | 119 筆（3 位排灣族長老） |
| 排灣語語料 | 117 句（問候/詢問/否定/文化） |
| 意圖訓練數據 | 158 筆 |
| 詞彙表擴展 | +104 tokens |
| API 端點 | 10 個 |

---

## 創新點

1. **低資源語言 ASR 微調** — 用 ≤200 筆語料即可訓練可用模型
2. **語言學知識注入** — 音韻規則引擎處理排灣語特有音變
3. **Chain-of-Thought 抗幻覺** — 每次回覆展示推理過程，可追溯、可驗證
4. **文化感知對話** — vuvu 角色不只是翻譯工具，而是文化傳承的載體
5. **可複製性** — 同一套框架可應用於台灣其他 15 族原住民語言
