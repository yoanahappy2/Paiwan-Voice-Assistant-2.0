# Demo 影片腳本 — 語聲同行 2.0

> 目標長度：60-90 秒
> 風格：情感故事 + 技術數據平衡
> 目標受眾：飛書 AI 挑戰賽評審

---

## 腳本

### 開場（0-10 秒）— 問題陳述

**畫面**: 黑底白字，逐行浮現

> 台灣 16 族原住民語言，全部瀕危。
> 排灣族年輕一代，正在失去母語。
> 如果 AI 能成為語言傳承的橋樑呢？

**旁白**（或文字）:
「聯合國說，全球 7000 種語言中 40% 正在消失。台灣排灣族語，就是其中之一。」

### 產品展示（10-45 秒）— 核心功能演示

**畫面**: Gradio Web 介面錄影

**場景 1: 語音問候（10-20s）**
- 點擊「🏠 問候」按鈕
- 畫面顯示: ASR 辨識 → RAG 檢索 → vuvu 思考過程 → 雙語回覆
- 🔊 vuvu 聲音播放（Grandma 語音）

**場景 2: 排灣語對話（20-35s）**
- 用戶輸入「masalu」（排灣語：謝謝）
- 展示 Chain-of-Thought 推理過程
- vuvu 回覆含排灣語 + 中文雙語

**場景 3: 文化故事（35-45s）**
- 點擊「📖 說故事」
- vuvu 分享排灣族文化知識
- 展示系統的文化理解能力

### 技術架構（45-60 秒）— 數據說話

**畫面**: 技術架構圖 + 數據表

> 技術架構:
> ASR (Whisper-tiny + LoRA, 81% 準確率)
> → RAG 檢索 (智譜 embedding-3 + FAISS, Top-3: 90%)
> → LLM (智譜 GLM-4-Flash, 命中率: 100%)
> → TTS 語音合成

**數據亮點**:
- 119 筆真實語料即可微調
- RAG vs Legacy: 命中率 100% vs 60%
- 端到端延遲 3-4 秒

### 願景（60-75 秒）— 社會意義

**畫面**: 排灣族文化意象 + 架構擴展示意圖

> 語聲同行 2.0 不只是一個 App。
> 它是一套可複製的瀕危語言 AI 框架。
> 同一套架構，可以延伸到台灣其他 15 族原住民語言。
> 未來，每個部落的孩子都能跟 AI vuvu 用母語對話。

### 結尾（75-90 秒）

**畫面**: 專案名稱 + GitHub QR Code

> 語聲同行 2.0
> Voice Companion for Endangered Languages
> GitHub: github.com/yoanahappy2/Paiwan-Voice-Assistant-2.0
> 讓每句排灣語，都有人聽見。

## 錄製檢查清單

### 前置準備
- [ ] ASR 服務啟動（`localhost:8000`）
- [ ] Gradio 啟動（`localhost:7860`）
- [ ] 螢幕錄影軟體準備（OBS / QuickTime）
- [ ] 麥克風測試
- [ ] 關閉通知（勿擾模式）

### 錄製注意
- 視窗大小固定，不要調整
- 字體放大（評審要看得清楚）
- 操作節奏放慢，每個步驟停留 2-3 秒
- TTS 播放時等聲音結束再繼續

### 剪輯
- 開頭和結尾加字幕
- 技術數據加 highlight 動畫
- 總長度控制在 60-90 秒
