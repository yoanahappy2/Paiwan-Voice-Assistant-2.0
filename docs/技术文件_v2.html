一、项目概述

"语声同行 2.0"是一套以排湾族语为对象所构建之 AI 智能教学系统。系统整合语音识别（ASR）、音韵校正、语法解释、语音合成（TTS）、语义检索（RAG）及测验评估六大核心模块，实现从语音输入到教学反馈的完整闭环，目前已完整部署至 Hugging Face Spaces 并可公开使用。

在线系统地址：https://huggingface.co/spaces/yoanahappy2/paiwan-voice-assistant

二、问题与动机

台湾 16 族原住民语言已全数被联合国教科文组织列为濒危语言。以排湾族为例，年轻一代面临严重的母语断层，日常生活中缺乏即时语音互动之学习场景；传统语言教学以课本与预录音频为主，无法提供对话式练习与即时纠正。

在全球尺度上，超过 7,000 种语言中约有 40% 正在消失。本项目希望透过 AI 技术，为濒危语言建立一套可复制、可扩展的数字保护与教学框架，以排湾族语为先导案例，验证技术可行性。

四、核心技术

4.1　ASR：Whisper-tiny + LoRA 微调

本研究采用 OpenAI Whisper-tiny（39M 参数）作为基座模型，透过 LoRA（Low-Rank Adaptation）进行参数高效微调，以排湾族长老之真实录音作为训练数据。相关超参数与成果如下表所示：

项目说明
基座模型openai/whisper-tiny（39M 参数）
微调方式LoRA（r=32, alpha=64）
训练数据120 笔排湾族长老真实录音
可训练参数量仅 2.3 MB（LoRA adapter）
ASR 识别准确率81.05%
经音韵校正后准确率约 90%

表 1：ASR 模型训练参数与成果

本实验验证了一项关键假设：在低资源语言场景下，不超过 200 笔语料即可训练出可用之 ASR 模型，为濒危语言的语音识别开辟了务实可行的路径。

4.2　音韵规则引擎

排湾语存在多种音变现象，尤其在老年发音者中更为显著。系统透过三层规则进行后处理校正：

  1. 特定词汇校正：处理高频误识别词，如 sima 校正为 siva、vasalu 校正为 masalu。
  2. m / v 音变互换：处理老年发音者特有的 mavan / vavan 交替现象。
  3. 舌尖音规则替换：处理 tj / t、lj / l 等舌尖音变化。

此音韵规则引擎将 ASR 输出之准确率由 81.05% 提升至约 90%，增幅达 9 个百分点。

4.3　RAG 语义检索

系统采用检索增强生成（Retrieval-Augmented Generation）架构，以动态语料匹配取代静态硬编码 prompt，其技术参数如下：

项目说明
Embedding 模型智谱 embedding-3（2048 维向量）
向量数据库FAISS IndexFlatL2
语料规模118 句（东排湾方言）
Top-3 命中率90%
Token 节省比例77%（相较于硬编码 prompt）

表 2：RAG 语义检索技术参数

4.4　LLM 语法解释

系统选用智谱 GLM-4-Flash 作为语法解释之语言模型。在架构设计上有一项核心原则：LLM 仅负责以中文进行语法分析与学习建议，不生成任何排湾语内容。此设计从根本上避免了低资源语言场景下常见的模型幻觉（hallucination）问题。

模型输出涵盖以下功能：词汇结构拆解、语法成分分析、学习策略建议、举一反三范例。

4.5　TTS 语音合成

系统采用 XTTS v2 模型实现零样本语音克隆（Zero-shot Voice Cloning），以排湾族长老之真实录音作为参考音色，合成出具有文化传承意义的 AI 语音输出。训练环境为清华大学算力平台（NVIDIA RTX 4090 GPU），目前已完成 50 个常用词汇与 8 个完整句子之语音合成。

4.6　缀词结构分析

排湾语属黏着语（Agglutinative Language），词汇由词根搭配多种前缀组合而成。系统可自动识别 11 种前缀标记，包含：

  • 时态标记：na（过去式）、uri（未来式）
  • 否定标记：ini（不）、inika（不是）、maya（不要）
  • 语态标记：ma（状态）、pa（致使）、si（相互）等

七、创新点

  1. 低资源语言 ASR 微调方案：证明不超过 200 笔语料即可训练出 81.05% 准确率之 ASR 模型，且该方案可复制至全球超过 3,000 种面临类似困境之濒危语言。
  2. 语言学知识注入：将排湾语音韵学知识编码为规则引擎，将 ASR 准确率由 81% 提升至约 90%，验证了语言学知识与深度学习模型互补的有效性。
  3. 抗幻觉架构设计：LLM 仅负责中文解释、不生成排湾语内容，搭配 RAG 动态语料匹配，从架构层面解决低资源语言之模型幻觉问题。
  4. 零样本语音克隆应用：以排湾族长老真实录音为参考音色，实现具文化传承意义的 AI 语音输出，重现即将消逝的族语长者声音。
  5. 端到端可复制框架：涵盖 ASR 至 TTS 之完整技术链路，架构设计具通用性，可推广至台湾其余 15 族原住民语言及全球其他濒危语言。

八、社会价值

  • 语言保存：为濒危语言提供即时互动式学习工具，降低语言消亡的速度。
  • 文化传承：AI vuvu（排湾语"祖父母"之意）角色使年轻一代得以在对话中同时学习族语与文化知识。
  • 教育平权：大幅降低原住民语言学习门槛，使每位学习者皆能拥有个人化 AI 语伴。
  • 可复制性：本框架可推广至台湾 16 族乃至全球濒危语言，具高度扩展潜力。

九、未来规划

  1. 飞书机器人整合：开发飞书 Bot 界面，使用户得以在飞书平台上直接与系统互动（目前已有架构设计原型）。
  2. 跨语言扩展：将框架推广至阿美族语、泰雅族语等其他台湾原住民族语言。
  3. 社区用户测试：与排湾族社区合作，收集真实使用者反馈以持续优化系统。
  4. 模型升级：尝试采用更大规模之 Whisper 模型，并扩充训练语料。
  5. 离线部署：支持边缘设备离线使用，使偏远地区亦能受益。

十、已知限制与讨论

本项目在低资源语言技术方面取得了阶段性成果，但仍有以下已知限制值得进一步探讨：

10.1　TTS 语音合成之发音准确度

XTTS v2 是目前开源社区中对低资源语言最为友善的语音合成模型之一，其零样本克隆能力仅需一段参考录音即可生成目标音色之语音，无需该语言之大规模语音数据集。然而，该模型并未原生支持排湾语（ISO 639-3: pag），系统实际上以英文模式（language code = "en"）驱动合成，导致部分词汇之发音与排湾语标准发音存在偏差。

此一限制反映了当前 TTS 技术在低资源语言场景下的普遍困境：全球超过 3,000 种濒危语言中，绝大多数缺乏足够的语音数据以训练专属 TTS 模型。未来可行的改进方向包括：（1）建立排湾语之国际音标（IPA）映射表，将排湾语拼字转换为 IPA 序列后再输入 TTS 模型；（2）收集更多排湾语语音数据以微调 TTS 模型之编码器；（3）探索基于语音合成标记语言（SSML）之音素级控制方案。

10.2　ASR 训练数据规模

本研究仅使用 120 笔录音进行 LoRA 微调即达到 81.05% 之准确率，经音韵校正后提升至约 90%。此结果虽已验证了低资源微调之可行性，但 90% 之准确率在实用场景中仍不足以完全替代人工校正。若要达到生产级品质（WER < 10%），仍需持续扩充语料规模。理想的训练数据量应达到 500–1,000 笔以上。

10.3　RAG 语料库覆盖范围

当前语料库包含 118 句东排湾方言语料，覆盖了日常对话之核心场景，但对于排湾语内部之方言变体（如东排湾、北排湾、南排湾之间之差异）尚未有完整覆盖。未来应纳入更多方言变体之语料，并引入方言识别模块以自动匹配对应之语言变体。

10.4　用户测试与验证

本系统目前尚未进行大规模用户测试。系统之实用性评估主要基于开发者自身之技术测试，缺乏排湾族社区使用者之真实反馈。未来应与排湾族社区及语言教师合作，进行系统性之用户测试，以验证系统在实际教学场景中之有效性。

10.5　小结

以上限制本质上均源于同一核心问题：低资源语言之数据稀缺。本项目的贡献不在于完全解决此问题，而在于希望提出一套务实、可复制的框架，在数据极度有限的条件下最大化 AI 技术之教学应用价值。随着更多语料之收集与模型技术之进步，目前面临的限制会逐步被克服。
