### 專案：語聲同行 2.0 — 排灣族語 AI 智能教學系統

**解決的問題**：台灣排灣族語被聯合國列為瀕危語言，全台 16 族原住民語言全部面臨傳承危機。年輕一代缺乏母語互動學習場景，傳統教學方式無法提供即時語音互動。全球 7000+ 種語言中 40% 正在消失，這是台灣的問題，也是全球性的文化危機。

**解決方案**：建立一套完整的排灣語 AI 教學系統，包含 7 大功能模組，並已部署上線（https://huggingface.co/spaces/yoanahappy2/paiwan-voice-assistant），評審可直接操作體驗。

**項目形態與內容**：

1. **發音評測系統**：學習者說排灣語 → Whisper-tiny + LoRA 微調 ASR 辨識 → 三層音韻規則校正（處理排灣語特有音變） → Levenshtein 距離評分 + 綴詞結構分析
2. **語法解釋器**：選擇任意排灣語句子 → LLM 進行詞彙拆解、語法結構分析、學習建議、舉一反三（LLM 只做中文解釋，不生成排灣語，避免幻覺）
3. **語料庫瀏覽**：117 句結構化排灣語語料，涵蓋問候/詢問/否定/文化等類別，附語法註解
4. **TTS 語音合成**：XTTS v2 零樣本語音克隆，以排灣族長老真實錄音為參考音色，合成 50 個常用詞 + 8 個完整句子
5. **RAG 語意檢索**：智譜 embedding-3 + FAISS 向量索引，118 筆語料向量化，支持中排雙語語意檢索，Top-3 命中率 90%
6. **測驗系統**：基於語料庫自動生成選擇題，即時批改
7. **訓練成果展示**：Whisper + LoRA 訓練 Loss 曲線 + 技術指標

**落地效果與量化產出**：

| 指標 | 數據 |
|------|------|
| ASR 準確率 | 81.05%（音韻校正後提升至 ~90%） |

## 🔴 項目中的 AI 技術方案有哪些？

### 1. ASR 語音辨識 — Whisper-tiny + LoRA 微調
- 基座模型：openai/whisper-tiny（39M 參數）
- 微調方式：LoRA（r=32, alpha=64, target=q_proj/v_proj），僅新增 2.3MB 可訓練參數
- 訓練數據：120 筆排灣族語錄音（3 位長老，真實場景）
- 結果：81.05% 準確率，證明 ≤200 筆語料即可訓練出可用的瀕危語言 ASR

### 2. 音韻規則引擎 — 語言學知識注入
- 三層校正：特定詞彙校正 → m/v 互換處理 → 音韻規則替換
- 處理排灣語特有音變（如 tj/t, lj/l, m↔v 混淆，老年發音者特徵）
- ASR 準確率從 81% → ~90%（規則覆蓋範圍內 100% 成功）

### 3. LLM 語法解釋 — 智譜 GLM-4-Flash
- 角色設定：排灣語教學助手
- 關鍵設計：LLM 只做中文解釋，不生成排灣語 → 從根本避免低資源語言幻覺
- Chain-of-Thought 推理過程展示，可追溯、可驗證

### 4. RAG 檢索增強生成 — 智譜 embedding-3 + FAISS
- 117 句語料 + 文化知識向量化（2048 維）
- 動態語意匹配取代硬編碼 prompt，Token 用量節省 77%
- 支持中文/排灣語跨語言檢索

### 5. TTS 語音合成 — XTTS v2 零樣本克隆
- 使用排灣族長老真實錄音作為參考音色
- 零樣本語音克隆，無需排灣語 TTS 訓練數據
- 部署於清華算力平台（NVIDIA RTX 4090）

### 6. 綴詞結構分析 — 排灣語黏著語分析器
- 11 種排灣語前綴自動識別（時態/否定/語態）
- 黏著語結構可視化展示

### AI 在項目中扮演的角色
AI 是整個系統的核心引擎：ASR 讓機器「聽懂」排灣語、LLM 讓機器「解釋」排灣語、TTS 讓機器「說出」排灣語、RAG 讓機器「找到」相關語料。這是一個端到端的 AI 應用，不是調用 API 的 wrapper——每個環節都有針對低資源語言的專門工程（LoRA 微調、音韻規則注入、幻覺防護、零樣本克隆）。

## 創新點

1. **低資源語言 ASR 微調** — 用 ≤200 筆語料即可訓練出可用模型（81.05%），證明瀕危語言不需要海量數據也能建立語音辨識，可複製到全球 3000+ 種瀕危語言
2. **語言學知識注入** — 音韻規則引擎處理排灣語特有音變，將 ASR 準確率從 81% 提升至 ~90%，是「AI + 語言學」的跨領域結合
3. **抗幻覺架構** — LLM 只做中文解釋、不生成排灣語，從根本解決低資源語言幻覺問題；RAG 動態匹配語料取代硬編碼
4. **零樣本語音克隆** — XTTS v2 以排灣族長老錄音為參考音色，實現文化傳承意義上的「AI vuvu（祖父母）聲音」
5. **端到端可複製框架** — 同一套架構可推廣至台灣其他 15 族原住民語言，從 ASR 到 TTS 全鏈路
