## 7. 曾经做过的最有代表性的AI项目/作品/研究*

**语声同行 2.0（Voice Companion 2.0）—— 排湾族语 Voice-to-Voice AI 语伴**

**解决的问题：**
台湾排湾族语被联合国列为濒危语言，全台16族原住民语言全部面临传承危机。年轻一代缺乏母语互动学习场景，传统教学（课本+录音）无法提供即时语音互动。

**项目内容：**
建立了一套完整的 Voice-to-Voice AI 系统：用户说排湾语 → ASR 语音辨识 → LLM 理解与生成 → 双语回复（排湾语+中文）。核心角色是 vuvu Maliq（排湾族AI祖母），能够双语对话、识别12类排湾语意图、纠正发音、分享文化知识，并通过 Chain-of-Thought 推理展示降低幻觉。

**技术方案：**
- ASR：Whisper-tiny + LoRA 微调，用 ≤200 笔语料训练出 81.05% 准确率
- 音韵规则引擎：处理排湾语特有音变（如 m↔v 混淆），将 ASR 准确率提升至 90%+
- LLM：智谱 GLM-4-Flash 驱动 vuvu Maliq 角色对话，117句排湾语语料注入
- 意图识别：12类意图 + 綴词法则，95.2% 分类准确率
- RAG 知识库：向量检索增强，动态匹配语料
- TTS：XTTS v2 微调排湾语语音合成（进行中）

**可量化产出：**
- ASR 准确率 81.05%（微调后 vs 微调前 ~30%，提升 170%）
- 音韵校正成功率 100%（规则覆盖范围内）
- 意图分类准确率 95.2%
- 真实录音语料 119 笔（3位排湾族长老录制）
- 排湾语语料库 117 句 + 104 个扩展 tokens
- GitHub 开源：https://github.com/yoanahappy2/Paiwan-Voice-Assistant-2.0

**项目形态：** Gradio Web 介面 + REST API，支持录音输入、文字输入、场景对话

## 8. 项目中的AI技术方案有哪些？*

本项目是一个完整的 AI 全链路系统，涵盖以下 AI 技术方案：

**1. 语音辨识（ASR）—— Whisper-tiny + LoRA 微调**
- 基于 OpenAI Whisper-tiny 模型，使用 LoRA（Low-Rank Adaptation）进行排湾语微调
- 仅用 119 笔真实录音语料（3位排湾族长老），训练出 81.05% 辨识准确率
- 证明了低资源语言（≤200笔数据）也能建立可用的语音辨识系统

**2. 音韵规则引擎**
- 基于排湾族语言学知识，构建音韵规则（如 tj→t, m↔v 混淆等）
- 结合 Levenshtein 距离进行模糊匹配
- 将 ASR 辨识准确率从 81% 提升至 90%+，噪音容忍度 96.2%

**3. 大语言模型（LLM）—— 智谱 GLM-4-Flash**
- 构建 vuvu Maliq（排湾族AI祖母）角色对话系统
- 注入 117 句排湾语语料 + 文化知识
- Chain-of-Thought 推理展示，降低低资源语言场景下的幻觉问题

**4. 意图识别**
- 12 类排湾语意图分类（问候、询问、否定、文化故事等）
- 基于綴词法则 + 158 笔训练数据
- 平均置信度 0.84，分类准确率 95.2%

**5. 检索增强生成（RAG）**
- 117 句语料向量化，动态检索匹配
- 替代硬编码 prompt，提升回复质量与灵活性

**6. 语音合成（TTS）—— XTTS v2 微调**
- 基于真实排湾族语音进行零样本/少样本语音克隆
- 扩展词彙表 +104 tokens，支持排湾语特有音节
