## 一、專案概述

**語聲同行 2.0** 是一套為瀕危的排灣族語建立的 AI 智能教學系統，涵蓋語音辨識、音韻校正、語法解釋、語音合成、語意檢索和測驗評估六大功能，已完整部署上線。

**線上 Demo**：https://huggingface.co/spaces/yoanahappy2/paiwan-voice-assistant

---

## 二、問題與動機

台灣 16 族原住民語言全部被聯合國列為瀕危語言。排灣族年輕一代面臨母語斷層，缺乏即時語音互動學習場景。傳統語言教學以課本和錄音為主，無法提供對話式練習。

全球 7000+ 種語言中 40% 正在消失，這不僅是台灣的問題，更是全球性的文化危機。本專案希望用 AI 技術，為瀕危語言建立一套可複製、可擴展的保護框架。

---

## 三、系統架構

```
用戶語音/文字
    ↓
ASR（Whisper-tiny + LoRA 微調）→ 排灣語文字辨識
    ↓
音韻規則引擎（三層校正）→ 處理排灣語特有音變
    ↓
RAG 檢索（智譜 embedding-3 + FAISS）→ 語意匹配語料庫
    ↓
LLM（智譜 GLM-4-Flash）→ 中文語法解釋 + 學習建議
    ↓
TTS（XTTS v2 零樣本克隆）→ 排灣族長老音色語音輸出
```

## 四、核心技術

### 4.1 ASR：Whisper-tiny + LoRA 微調

| 參數 | 值 |
|------|-----|
| 基座模型 | openai/whisper-tiny（39M 參數） |
| 微調方式 | LoRA（r=32, alpha=64） |
| 訓練數據 | 120 筆排灣族長老真實錄音 |
| 可訓練參數 | 僅 2.3MB（LoRA adapter） |
| ASR 準確率 | 81.05% |
| 校正後準確率 | ~90% |

**創新**：證明 ≤200 筆語料即可訓練出可用的瀕危語言 ASR 模型。

### 4.2 音韻規則引擎

三層校正處理排灣語特有音變：

1. **特定詞彙校正**：如 sima→siva, vasalu→masalu
2. **m/v 互換**：老年發音者特徵（如 mavan↔vavan）
3. **音韻規則替換**：tj/t, lj/l 等舌尖音變化

### 4.3 RAG 語意檢索

| 參數 | 值 |
|------|-----|
| Embedding 模型 | 智譜 embedding-3（2048 維） |
| 向量庫 | FAISS IndexFlatL2 |
| 語料規模 | 118 句（東排灣方言） |
| Top-3 命中率 | 90% |
| Token 節省 | 77%（vs 硬編碼 prompt） |

### 4.4 LLM 語法解釋

- 模型：智譜 GLM-4-Flash
- **關鍵設計**：LLM 只做中文解釋，不生成排灣語 → 從根本避免低資源語言幻覺
- 功能：詞彙拆解、語法結構分析、學習建議、舉一反三

### 4.5 TTS 語音合成

- 模型：XTTS v2（零樣本語音克隆）
- 參考音色：排灣族長老真實錄音
- 訓練環境：清華算力平台（NVIDIA RTX 4090）
- 成果：50 個常用詞 + 8 個完整句子

### 4.6 綴詞結構分析

排灣語是黏著語（agglutinative），系統自動識別 11 種前綴：

- 時態標記：na（過去）、uri（未來）
- 否定標記：ini（不）、inika（不是）、maya（不要）
- 語態標記：ma（狀態）、pa（致使）、si（相互）等

## 七、創新點

1. **低資源語言 ASR 微調** — ≤200 筆語料訓練出 81.05% 準確率，可複製至全球 3000+ 種瀕危語言
2. **語言學知識注入** — 音韻規則引擎將 ASR 準確率從 81% 提升至 ~90%
3. **抗幻覺架構** — LLM 只解釋不生成 + RAG 動態匹配，從根本解決低資源語言幻覺
4. **零樣本語音克隆** — 以排灣族長老錄音為參考音色，實現文化傳承意義的 AI vuvu 聲音
5. **端到端可複製框架** — 從 ASR 到 TTS 全鏈路，可推廣至台灣其他 15 族原住民語言
