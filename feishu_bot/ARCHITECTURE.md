## 整合方式

### 方案 A：飛書機器人（推薦）
- **觸發方式**: 用戶在飛書群組或私聊 @機器人
- **支援格式**: 文字訊息（直接處理）/ 語音訊息（先轉文字再處理）
- **回覆格式**: 文字（排灣語 + 中文雙語）

### 方案 B：飛書小程式
- 嵌入式 Web 介面，體驗更接近原生
- 需要前端開發，時程較長

### 方案 C：飛書文件機器人
- 在飛書文檔中插入智能模塊
- 適合教學場景

## 訊息流程

### 文字訊息
```
用戶: "masalu"
  → Bot 收到文字
  → RAG 檢索 "masalu" → 匹配到 "masalu! = 謝謝！"
  → LLM 生成回覆
  → 回覆: "ai~~ masalu aravac! 謝謝你孩子！"
```

### 語音訊息
```
用戶: [語音] "na tarivak sun?"
  → Bot 收到音頻 file_key
  → 下載音頻 → ASR 辨識 → "na tarivak sun?"
  → RAG 檢索 → LLM 生成
  → 回覆: "na tarivak aken! 你好嗎？vuvu 很好呢！"
```

## API 端點設計

| 端點 | 方法 | 說明 |
|------|------|------|
| `/feishu/webhook` | POST | 飛書事件回調 |
| `/feishu/command` | POST | 飛書指令（如 /learn, /quiz） |
| `/health` | GET | 健康檢查 |

## 擴展功能（入營後實作）

1. **每日排灣語**: 每天早上自動推送一句排灣語
2. **互動測驗**: `/quiz` 指令觸發排灣語小測驗
3. **語音教學**: 發送語音 → vuvu 糾正發音 → 回傳正確讀法
4. **文化卡片**: 圖文並茂的排灣族文化知識推送
5. **群組互動**: 多人在群組中跟 vuvu 學排灣語
