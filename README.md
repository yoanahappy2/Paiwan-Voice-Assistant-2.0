# 語聲同行 2.0 — 瀕危語言 AI 保護框架

> **Agent-based 模組化架構** | **研究語言**：東排灣族語（Paiwan, ISO 639-3: pag）
> 在語料極度稀缺（120 筆）的條件下，構建完整的瀕危語言 AI 教學系統

## 專案概述

全球 7,000 種語言中超過 40% 面臨瀕危，主流 AI 模型（GPT-4、GLM-4）對排灣語的零樣本翻譯 BLEU 為 **0.0**——完全不懂。本專案提出一套**可遷移的瀕危語言 AI 保護框架**，在極少語料（100-500 句）下即可構建可用的語音辨識、翻譯、教學和語料增長服務。

### 核心成果

| 模組 | 指標 | 數據 |
|------|------|------|
| ASR（自訓練） | 準確率 | 30% → **96.8%**（Whisper-small + LoRA，120 筆語料） |
| 翻譯（RAG） | BLEU | 0.0 → **0.916**（4,844 條語料，混合檢索） |
| TTS | 語音覆蓋 | **151 詞**全部有語音（101 真人 + 28 TTS 驗證通過 + 22 備用） |
| 飛書 Bot | 指令數 | **9 個**（翻譯、教學、測驗、口語練習、每日一句等） |
| 知識圖譜 | 詞彙關聯 | **47 詞**雙向關聯 + 10 條綴詞規則 |

## 技術貢獻

### 1. 極少語料 ASR 微調
- **Whisper-small（244M 參數）+ LoRA（rank=32, alpha=64）**，僅 14MB adapter
- 120 筆排灣族長老真實錄音，準確率從 30% 提升至 **96.8%**
- 三層音韻校正引擎，校正成功率 100%

### 2. RAG 混合檢索翻譯
- **關鍵詞檢索（×2 權重）+ 向量檢索（智譜 embedding-3 + FAISS，×1 權重）**
- 4,844 條雙語語料，覆蓋 13 類排灣語內容
- ICL 對比實驗：5/10/20-shot BLEU 全為 0，RAG 達 **0.916**

### 3. 分層語音策略
- **Layer 1**：真人母語者錄音（101 詞，最高品質）
- **Layer 2**：XTTS v2 合成 + ASR 反向驗證通過（28 詞）
- **Layer 3**：XTTS v2 合成未通過驗證（22 詞，備用降級）
- 151 詞全部有語音覆蓋

### 4. 主動學習閉環
- 用戶翻譯確認 → 語料回流 → 語料庫自動增長
- `active_learner.py` + `retrain_pipeline.py`
- 社群語料審核：用戶提交 → 母語者審核 → 回流語料庫

### 5. 飛書 Bitable 數據中控台
- 學習熱力圖：追蹤每個用戶的學習軌跡
- 社群語料審核表：母語者審核用戶提交的新詞
- 詞彙掌握統計：量化學習效果

### 6. 知識圖譜引擎
- 10 個前綴規則 + 13 個親屬稱謂 + 7 個語義群組
- 47 條詞彙雙向關聯，支持 `/lookup` 查詢

## 可遷移性

本框架設計為可遷移到其他瀕危語言：

1. **ASR 模組**：只需 100-500 筆錄音即可微調 Whisper + LoRA
2. **RAG 翻譯**：替換語料庫即可，無需重新訓練 LLM
3. **音韻引擎**：替換為目標語言的音韻規則
4. **Bitable 中控台**：飛書原生能力，零額外開發
5. **主動學習**：用戶使用過程自動累積語料

潛在推廣目標：阿美語、泰雅語、布農語等台灣原住民語言，乃至全球 3,000+ 瀕危語言。
