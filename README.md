# 語聲同行 2.0 — 可遷移的瀕危語言 AI 保護框架

> 用 120 句訓練出 96.8% 準確率的 ASR，用 200 句建立 BLEU 0.916 的翻譯服務——為全球 3,000+ 種瀕危語言提供可遷移的 AI 保護方案。
>
> **研究語言**：東排灣族語（Paiwan, ISO 639-3: pag）

## 專案概述

全球 7,000 種語言中超過 40% 面臨瀕危，主流 AI 模型（GPT-4、GLM-4）對排灣語的零樣本翻譯 BLEU 為 **0.0**——完全不懂。本專案提出一套**可遷移的瀕危語言 AI 保護框架**，在極少語料（100-500 句）下即可構建可用的語音辨識、翻譯、教學和語料增長服務。

### 核心成果

| 模組 | 指標 | 數據 |
|------|------|------|
| ASR（自訓練） | 準確率 | 30% → **96.8%**（Whisper-small + LoRA，120 筆語料） |
| 翻譯（RAG） | BLEU | 0.0 → **0.916**（4,844 條語料，混合檢索） |
| TTS | 語音覆蓋 | 分層策略：**119 詞真人錄音** + XTTS v2 合成，ASR 反向驗證品質 |
| 飛書 Bot | 指令數 | **9 個**（翻譯、教學、測驗、口語練習、每日一句等） |
| 知識圖譜 | 詞彙關聯 | **47 詞**雙向關聯 + 10 條綴詞規則 |

每個 Agent 將**語言學規則**（音韻引擎、綴詞分析）注入 AI 流程中，解決低資源語言的幻覺問題。所有 Agent 共享 RAG 知識庫與 Bitable 數據中控台，形成「用戶使用 → 數據回流 → 模型優化」的完整閉環。

**已驗證的數據飛輪閉環**（端到端測試通過，2026-05-07）：用戶在飛書 Bot 對話 → 高置信度翻譯自動寫入 Bitable → 同步腳本增量拉取 → 更新 HF Space 語料庫 → 線上 Demo 即時展示社群貢獻。語料從 42 句增長至 53 句（含 11 條社群貢獻），每次執行 `sync_bitable_to_hf.py --push` 即可增量同步。

## 技術貢獻

### 1. 極少語料 ASR 微調
- **Whisper-small（244M 參數）+ LoRA（rank=32, alpha=64）**，僅 14MB adapter
- 120 筆排灣族長老真實錄音，準確率從 30% 提升至 **96.8%**
- 三層音韻校正引擎，校正成功率 100%

### 2. RAG 混合檢索翻譯
- **關鍵詞檢索（×2 權重）+ 向量檢索（智譜 embedding-3 + FAISS，×1 權重）**
- 4,844 條雙語語料，覆蓋 13 類排灣語內容
- ICL 對比實驗：5/10/20-shot BLEU 全為 0，RAG 達 **0.916**

### 3. 分層語音策略
- **Layer 1**：真人母語者錄音（119 詞，最高品質，排灣族長老親口錄製）
- **Layer 2**：XTTS v2 合成 + ASR 反向驗證通過
- **Layer 3**：XTTS v2 合成未通過驗證（備用降級）
- 關鍵設計：用自訓練 ASR 當 TTS 質檢員——如果 ASR 都能聽懂合成語音，說明發音可接受

### 4. 語音閉環：聽→說→辨識→評價
完整的口語練習閉環（`/speak` 指令）：聽真人/TTS 參考發音 → 用戶錄音 → 自訓練 Whisper-small + LoRA（96.8%）即時辨識 → 音韻相似度 + 綴詞分析 + 分數評級
- 突破點：主流語言學習 App 調用通用 ASR API，對排灣語完全無效。我們用自訓練模型實現了「專屬語言老師」級別的口語評測

### 5. 主動學習閉環 + 數據飛輪
- 用戶翻譯確認 → 語料回流 → 語料庫自動增長
- `active_learner.py` + `retrain_pipeline.py` + `sync_bitable_to_hf.py`
- 社群語料審核：用戶 `/submit` 提交 → 母語者在 Bitable 審核 → 通過後回流語料庫
- 端到端驗證通過：語料從 42 句增長至 53 句（含社群貢獻）

### 6. Bitable 對話驅動的社群語言保存平台
不只是記錄器，是活的協作平台：
- 學習熱力圖：追蹤每個用戶的學習軌跡，驅動個人化推薦
- 社群語料審核表：用戶 `/submit` 提交新詞 → 母語者直接在 Bitable 審核 → 通過後回流語料庫
- 詞彙掌握統計表：匯總所有用戶測驗結果，量化每個詞的掌握率
- 核心創新：母語者不需要學新工具，直接在飛書多維表格裡審核，像用 Excel 一樣簡單

### 7. 知識圖譜引擎
- 10 個前綴規則 + 13 個親屬稱謂 + 7 個語義群組
- 47 條詞彙雙向關聯，支持 `/lookup` 查詢

## 可遷移性

本框架設計為可遷移到其他瀕危語言：

| 組件 | 遷移成本 | 說明 |
|------|---------|------|
| Whisper + LoRA | 收集 100-200 句錄音 | 任何有文字系統的語言 |
| 音韻規則引擎 | 語言學家合作 1-2 天 | 編寫目標語言規則 |
| RAG 翻譯 | 整理 200-500 句雙語對照 | 任何有書面材料的語言 |
| 主動學習閉環 | 零改動 | 框架自帶 |
| **整體遷移** | **1-2 週** | **適配新語種** |

擴展計劃：短期穩定飛書 Bot + 擴大用戶測試 → 中期擴展至阿美語、泰雅語 → 長期覆蓋台灣 16 族原住民語言 → 願景：開源框架供全球瀕危語言社區使用。
